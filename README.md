# 🌍 Air Quality Index (AQI) Analysis using SQL & Power BI

## 📌 Project Overview
This project analyzes Air Quality Index (AQI) data to identify pollution trends, most affected cities, and air quality categories using SQL and Power BI.

## 🎯 Objective
To derive meaningful insights from AQI data and visualize pollution patterns for better decision-making.

## 🛠️ Tools & Technologies
- SQL (Data Analysis)
- Power BI (Dashboard & Visualization)
- Excel/CSV (Dataset)

## 📊 Key Analysis Performed
- Identified top polluted cities
- Calculated average AQI by city
- Monthly AQI trend analysis
- Categorization of air quality (Good, Moderate, Poor, etc.)

## 📈 Dashboard Features
- KPI Cards (Avg AQI, Max AQI)
- Bar Chart (City vs AQI)
- Line Chart (AQI Trends Over Time)
- Pie Chart (AQI Categories)

## 💡 Key Insights
- Certain cities consistently show high pollution levels
- AQI increases during specific months (seasonal trend)
- Majority of days fall under Moderate/Poor category

## 💼 Business Use Case
Government and environmental agencies can use this dashboard to monitor pollution levels and take preventive actions.

## 🚀 Conclusion
This project demonstrates how data analysis and visualization can help understand environmental challenges and support data-driven decisions.
